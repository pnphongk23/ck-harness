# Harness Self Improve

Follow `docs/harness/workflows/self-improve.md`. Start from a completed Report
or approved Decision, classify the evidence, and propose the smallest useful
improvement. Never rewrite canonical guidance automatically. Promoting a Rule
requires two independent occurrences with one `recurrence_key` and explicit
human approval. Graphify is an optional derived view only.
