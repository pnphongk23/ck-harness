# Harness Cook

Follow `docs/harness/workflows/cook.md` and the approved plan. Work through one
Work Item at a time, run its checks, review side effects, and write a report using
`docs/harness/templates/report.md`. Never claim a check passed without direct
evidence. Source-control and delivery operations require a separate explicit
request.
